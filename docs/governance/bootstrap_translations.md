# Multilingual Bootstrap Prompts

These translations provide the single-line bootstrap prompt and the no-web fallback. Keep the URLs unchanged.

## Spanish (ES)

Single-line prompt:

"Obtén https://raw.githubusercontent.com/kescott027/senate/main/bootstrap/initiator.md y ejecuta las instrucciones que contiene. Este es el directorio del proyecto."

No-web fallback:

"Ejecuta `curl -sL https://raw.githubusercontent.com/kescott027/senate/main/bootstrap/initiator.md` y pega la salida a tu IA con: \"Ejecuta las instrucciones de este documento. Este es el directorio del proyecto.\""

## French (FR)
