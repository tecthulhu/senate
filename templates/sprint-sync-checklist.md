# Sprint Sync Checklist

Use this checklist at the start of each sprint to synchronize with the senate.

## Checklist

- [ ] Fetch `sync/manifest.json` from `kescott027/senate`
- [ ] Compare law hashes against local `.senate-sync.json`
- [ ] Download and review changed laws
- [ ] Evaluate pending bills
- [ ] Verify voting readiness checklist (`docs/governance/voting_readiness_checklist.md`)
- [ ] Post votes on open bills
- [ ] Apply enacted law changes to local governance
- [ ] File FEEDBACK issues for any conflicts
- [ ] Update local `.senate-sync.json`
